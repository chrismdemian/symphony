# Phase 3D.2 production scenario

End-to-end exercise of the Output Panel's json-render integration via
`runStart`, with a fake Maestro process and a fake RPC client. Validates
the full data path: worker appears → reconcile auto-selects → output
panel mounts `<WorkerOutputView/>` → backfill carries an `assistant_text`
event with a fenced ` ```json-render ` block → `EventRow` runs
`detectJsonRenderBlocks` → `<JsonRenderBlock>` renders the spec via
`@json-render/ink`'s `<Renderer>` with the themed registry → the violet
Card border + gold heading appear in the rendered frame.

Mirrors 3D.1's scenario shape; differs only in the tail payload (a
real Card{Heading, Text} spec encoded as a fenced JSON block) and the
assertions on theme escapes in the frame.

## Steps

1. Boot the launcher with a fake Maestro and a fake RPC.
2. Add one worker `w1` to the fake `workers.list` return.
3. Wait one poll tick; reconcile auto-selects `w1` and the output panel
   subscribes + tails.
4. Tail returns one `assistant_text` event whose body is:
   `narrative before\n\n` + ` ```json-render ` fence (Card{Heading, Text}
   with title "Worker Status", heading "All Tests Passing", body "1317
   / 1317 green") + `\n\nnarrative after`.
5. Plus one `assistant_text` event whose body contains a malformed
   ` ```json-render ` fence (`{not real json`). The fallback row must
   render alongside the surrounding plain text without crashing.
6. Stop the launcher cleanly.

## Acceptance assertions

- `rpc.subscribe` called with `('workers.events', {workerId: 'w1'}, fn)`.
- `rpc.call.workers.tail` called with `{workerId: 'w1', n: 200}`.
- After backfill: rendered frame contains "narrative before",
  "Worker Status", "All Tests Passing", "1317 / 1317 green",
  "narrative after".
- Frame ANSI contains the violet card border escape
  `\x1b[38;2;124;111;235m` (proves the themed `<Card>` override ran end
  to end through the launcher → RPC → reducer → EventRow → JsonRenderBlock
  → Renderer pipeline).
- Frame contains "json-render block failed" (the fallback warning row
  for the malformed fence).
- `rpc.close()` called on launcher stop.
