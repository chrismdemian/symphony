# Phase 3D.1 production scenario

End-to-end exercise of the Output Panel via `runStart`, with a fake
Maestro process and a fake RPC client. Validates the full data path:
worker appears in the workers panel → `WorkerSelectionProvider` reconciles
auto-selects it → `<OutputPanel rpc={rpc}/>` mounts `<WorkerOutputView/>`
keyed by the selected id → the hook calls `rpc.subscribe('workers.events',
{workerId})` and `rpc.call.workers.tail({workerId, n: 200})` →
backfill + live events appear in the rendered TUI frame.

## Steps

1. Boot the launcher with a fake Maestro and a fake RPC (in-process,
   no real WebSocket).
2. Initial frame: zero workers; output panel shows "Select a worker —
   j/k or 1-9 in the workers panel.".
3. Mutate the fake RPC's `workers.list` return to one worker `w1`. Wait
   one poll tick (~1 s).
4. After the poll: `w1` shows up in the workers panel; reconcile
   auto-selects it; the output panel switches from the empty hint to
   `WorkerOutputView`. Subscribe + tail fire.
5. Tail returns a scripted sequence: `assistant_text` → `tool_use(Read,
   src/foo.ts)` → `tool_result(file body)`. Verify the rendered frame
   contains the assistant text, the `▸ Read  src/foo.ts` row, and the
   tool result body.
6. Emit a live `system_api_retry` event. The sticky amber banner row
   appears at the top of the panel (in addition to the inline retry row
   in the body).
7. Emit a live `assistant_text` event. The banner clears (still exactly
   one inline retry row in the body — audit trail preserved).
8. Stop the launcher cleanly.

## Acceptance assertions

- Empty-state hint visible BEFORE worker appears.
- `▸ Read  src/foo.ts` AND the file body visible AFTER tail backfill.
- "rate limited" string count = 2 right after retry (sticky banner +
  inline row), then = 1 after the follow-up text arrives (banner
  cleared, inline row remains).
- `rpc.subscribe` called with `('workers.events', {workerId: 'w1'}, fn)`.
- `rpc.call.workers.tail` called with `{workerId: 'w1', n: 200}`.
- `rpc.close()` called on launcher stop.
